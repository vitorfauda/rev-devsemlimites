# Dev Sem Limites — Painel de Revenda

Painel de revenda da extensão Dev Sem Limites. Hospedado em `rev.devsemlimites.site`.

## Stack
- Vite + React 18 + TypeScript
- Tailwind CSS + design system holográfico
- React Router v6, TanStack Query
- Supabase (auth + data + edge functions)
- @react-three/fiber + drei (hero 3D)
- Framer Motion, React Hook Form + Zod

## Dev
```bash
npm install
npm run dev
```

## Env vars
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
